/**
 * 欢迎来到CSS世界！本文将带领您学习CSS里30个必须记住的选择器，让我们开始吧......
 */

/**
 * 1、* ：在我们看比较高级的选择器之前，应该认识下这个众所周知的清空选择器。
 * 星号呢会将页面上所有每一个元素都选到。许多开发者都用它来清空内外边距。
 */
* {
  margin:0;
  padding:0;
}

/**
 * *也可以用来选择某元素的所有子元素。
 */
#container * {
  -webkit-transition:1s;
}

/**
 * 2、ID选择器 ：在选择器中使用#可以用id来定位某个元素。
 * id选择器是很严格的且没法复用。如果可以，首先试试用标签名字，或者是伪类。
 */
#c2 {
  height:45%;
}

/**
 * 3、类选择器 ：跟ID选择器不同的是，它可以定位多个元素。
 * 当你想对多个元素进行样式修饰的时候就可以使用class选择器。
 */
.leftContainer {
  height:45%;
}

/**
 * 4、标签选择器 X：定位页面上所有的某标签。
 * 如果希望定位某父节点下的所有某类标签，写法为：#c2 a
 */
a {
  color:white;
}

/**
 * 5、X:link 和 X:visited ：
 * link这个伪类来定位所有还没有被访问过的链接。
 * visited来定位所有已经被访问过的链接
 */
a:link {
  color:orange;
}
a:visited {
  color:red;
}

/**
 * 6、相邻选择器 X+Y ：选中指定元素的直接后继元素
 */
#c2 a+p {
  text-indent:40px;
}

/**
 * 7、子选择器 X>Y ：选中指定元素的直接子元素
 */
#c2>p {
  font-family:italic;
}

/**
 * 8、兄弟节点组合选择器 X~Y ：X+Y只会选择紧挨跟着指定元素的那些元素;
 * 而这个选择器，会选择跟在目标元素后面的所有匹配的元素
 */
#c2 a~p {
  font-family:italic;
}

/**
 * 9、属性选择器 X[title] ：下面例子中，只会选择有title属性的元素
 */
a[title] {
  color:green;
}
a[title="tss"] {
  font-size:16px;
}
a[title*="tss"] {
  text-decoration:underline;
}
a[href^="http"] {
  text-decoration:underline;
}
a[href$=".png"] {
  text-decoration:underline;
}

/**
 * 10、自定义属性选择器 X[data-*="tss"], 又叫钩子
 */
a[data-filetype="image"] {
  font-family:italic;
}

/**
 * 11、伪类选择器之 X:checked：定位那些被选中的单选框和多选框
 */
input[type=radio]:checked {
  border:1px solid black;
}

/**
 * 12、伪类选择器之 X:after、X:before：会在被选中的标签周围生成一些内容X:before
 * 下面这段代码会在目标标签后面补上一段空白，然后将它清除。
 */
.clearfix:after {
  content:"";
  display:block;
  clear:both;
  visibility:hidden;
  font-size:0;
  height:0;
}

/**
 * 13、伪类选择器之 X:hover，在用户鼠标飘过的地方涂点儿彩，这个伪类可以办到
 */
pre:hover {
  box-shadow:0px 0px 40px 5px rgba(255,255,255,0.4);
}

/**
 * 14、X:not(selector)， 如：*:not(p) 选中所有出段落标签之外的所有标签
 * 取反伪类很有用，下面写法把除id为container之外的所有div标签都选中。
 */
div:not(#container) {
  color:blue;
}

/**
 * 15、X:nth-child(n) 和 X:nth-last-child(n)， n为一个整形参数，从1开始。
 * 如果你想获取第二个元素那么你传的值就是li:nth-child(2)
 */
li:nth-child(3) {
  color:red;
}

/**
 * 16、X:nth-of-type(n) 和 X:nth-last-of-type(n)， n为一个整形参数，从1开始。
 * 根据元素的类型来进行选择，下面例子对#c2里第二个a元素进行修饰
 */
a:nth-of-type(2) {
  color:red;
}

/**
 * 17、X:first-child 和 X:last-child， 可以选择到第一个或最后一个子标签
 */
a:first-child, a:last-child {
  color:red;
}
